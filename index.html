Shortest path
The shortest path tree problem on directed graph.
Input:
  • A input directed graph G=(V,E)
  • A function w:E\to \R^+.
  • a vertex s\in V, the root vertex of the shortest path tree.
Output:
  • The shortest path tree rooted at vertex s
Complexity measure:
n
the number of vertices of G.
m
the number of edges of G.
The dijkstra algorithm using fibonacci heap.
  • time: O(m+n\log n)
Meet of two partitions
Input:
  • \mathcal{A} and \mathcal{B} partitions of [n].
Output:
  • The meet of \mathcal{A} and \mathcal{B}, \mathcal{C}=\set{A\cap B \mid A\in \mathcal{A}, B\in \mathcal{B}}.
Complexity measure:
n
The size of the universe.
index the sets in \mathcal{A} and \mathcal{B}. For each element i, find a_i and b_i where a_i is index of a set in \mathcal{A} that contains i, similarly for b_i. Group by (a_i,b_i) using radix sort.
  • time: O(n)
For each set in \mathcal{A}, interset it with all sets in \mathcal{B} through a partition refinement data structure.
  • time: O(n)

min-cost flow (integer cost and capacity)
Given a graph with edge capacity and cost, find the min-cost flow.
Input:
  • v\in \N the value of the flow.
  • The source and sink vertices s and t.
  • The capacity function, u:E\to \N
  • The cost function c:E\to \N
  • A input directed graph G=(V,E)
Output:
  • A minimum cost st-flow f:E\to \R of value v.
Complexity measure:
C
upper bound on the absolute value of the cost
n
number of vertices of G.
m
number of edges of G.
U
upper bound on the absolute value of the capacity
m'
number of edges with infinite capacity.
Edmonds and Karp
  • time: O((n+m') \log U (m+n \log n))
Ahuja, Goldberg, Orlin and Tarjan 1998
  • time: O(nm \log (n^2/m) \log (nC))
Orlin's 1991 paper
  • time: O((n+m') \log n S(m,n)) time. Here S(m,n) is running time for shortest path algorithm.
Goldberg and Tarjan 1987
  • time: O(nm \log (n^2/m) \log (nC))
  • min-cost flow (integer cost)
  • min-cost flow (integer capacity)
