Partition refinement
Maintain a partition \mathcal{X} of [m], such that taking a input set S, it creates a new partition \mathcal{X}_S = \set{ X\cap S, X\setminus S |X\in \mathcal{X} }.
We want O(n) time per operation. This is the partition refinement data structure.

Given u and a u-normal eventually periodic string s, find the the minimum split (a,b) of s.
An infinite string is given as a function, s:\N\to \Sigma. It take an index and output the character at the index. We say (a,b) is a split of s if ab=s and b is a infinite periodic string. A split (a,b) is minimum, if period of b is minimum and under the previous constraint, length of a is minimum. An eventually periodic string s is u-normal, if for any two indices i<j, where j-i\leq u and s[i..i+u] = s[j..j+u], there exists a split (a,b) of s where |a|\leq i, and the period of b is at most u.
