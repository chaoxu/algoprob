The h-index problem
Given positive integers a_1,\ldots, a_n, find the largest h such that |\set{i | a_i \geq h}|\geq h.
Replace each a_i with \min(a_i,n), and sort the new list.
  • Integer sorting
O(n) time
  • The weighted h-index problem
The weighted h-index problem
Given pairs (w_1,a_1),\ldots,(w_n,a_n), find the largest h such that \sum_{i:a_i\geq h} w_i \geq h.
Select weighted median, remove half of the elements, and recur.
  • Unordered weighted selection
O(n) time
  • The h-index problem : set all a_i to 1.
Meet of two partitions
Let \mathcal{A} and \mathcal{B} to be partitions of [n]. Compute the meet of the two partitions. \mathcal{C}=\set{A\cap B \mid A\in \mathcal{A}, B\in \mathcal{B}}.
index the sets in \mathcal{A} and \mathcal{B}. For each element i, find a_i and b_i where a_i is index of a set in \mathcal{A} that contains i, similarly for b_i. Group by (a_i,b_i) using radix sort.
  • Integer sorting
O(n) time
For each set in \mathcal{A}, interset it with all sets in \mathcal{B} through a partition refinement data structure.
  • Partition refinement
O(n) time
Partition refinement
Maintain a partition \mathcal{X}, such that taking a input set S, it creates a new partition \mathcal{X}_S = \set{ X\cap S, X\setminus S |X\in \mathcal{X} }.
We want O(|S|) time per operation. This is the partition refinement data structure.

Given u and a u-normal eventually periodic string s, find the the minimum split (a,b) of s.
An infinite string is given as a function, s:\N\to \Sigma. It take an index and output the character at the index. We say (a,b) is a split of s if ab=s and b is a infinite periodic string. A split (a,b) is minimum, if period of b is minimum and under the previous constraint, length of a is minimum. An eventually periodic string s is u-normal, if for any two indices i, where j-i\leq u and s[i..i+u] = s[j..j+u], there exists a split (a,b) of s where |a|\leq i, and the period of b is at most u.
